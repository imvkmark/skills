# Rust 项目分析要点

针对 Rust 项目（Actix, Rocket, Axum 等）的深度分析指导。

## 项目类型检测

**特征文件**:
- `Cargo.toml`
- `Cargo.lock`

**关键依赖识别**:
- Actix-web / Rocket / Axum (Web 框架)
- Tokio (异步运行时)
- SQLx / Diesel (ORM/SQL)
- Serde (序列化)
- Tracing (日志和追踪)

## 架构模式识别

### 单体架构特征
- 单一的 `Cargo.toml`
- 统一的项目目录结构
- 共享的数据库配置

### 微服务架构特征
- 多个独立的 crate
- 每个 crate 有独立的 `Cargo.toml`
- 服务间通过 HTTP/gRPC 通信

## 核心分析要点
