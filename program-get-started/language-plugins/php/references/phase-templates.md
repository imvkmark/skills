# PHP 学习阶段模板

PHP 知识点学习的标准阶段划分模板，根据不同的学习领域和目标提供结构化的知识点组织方式。

### 粒度划分标准
采用**五级粒度**的结构化组织方式，从宏观到微观逐级深入：

| 级别 | 名称 | 说明 | 示例 |
|------|------|------|------|
| 1 | 阶段（Phase） | 大的知识领域划分，对应学习路径的主要里程碑 | PHP 基础、PHP 高级特性、Web开发、微服务架构 |
| 2 | 模块（Module） | 阶段内的主题分组，相关知识点的集合 | 数据结构、面向对象、Laravel框架、性能优化 |
| 3 | 章节（Chapter） | 模块内的逻辑单元，独立的学习主题 | 数组操作、设计模式、ORM模型、缓存系统 |
| 4 | 知识点（Knowledge Point） | 章节内的具体概念，核心学习单元 | 数组函数、依赖注入、Eloquent查询、Redis使用 |
| 5 | 子知识点（Sub Point） | 知识点的细分细节，具体的技术点 | 数组排序函数、IOC容器原理、关联模型、缓存击穿解决方案 |

> 灵活使用原则：简单知识点可省略部分层级，避免过度拆分增加认知负担。入门学习使用3级（阶段→模块→知识点）即可，中高级/面试学习使用完整5级结构。
